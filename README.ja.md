# tview-tutorial

## 前提条件

このアプリケーションを実行するために必要なもの：

- Docker

## Usage

1. Dockerコンテナの起動  
   Docker Composeを使用して、必要な環境をセットアップします。  
   以下のコマンドを実行して、バックグラウンドでコンテナを起動します。
    ```bash
    $ docker-compose up -d
    ```

2. アプリケーションのビルド
    次に、Makefileを使用してアプリケーションをビルドします。以下のコマンドを実行します。
    ```bash
    $ make build
    ```

3. アプリケーションの実行
    ビルドが完了したら、以下のコマンドでアプリケーションを実行します。

    ```bash
    $ make run
    ```
